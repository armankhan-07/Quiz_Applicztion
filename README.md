# 🎯 Quiz Application

A responsive and interactive Quiz Application built using **HTML, CSS, and JavaScript**. The application allows users to answer multiple-choice questions, receive instant feedback, track their progress, and save their highest score using the browser's Local Storage.

---

## 📖 About the Project

This project is designed to demonstrate core front-end web development concepts without using any frameworks.

The application presents one question at a time, validates the user's answer instantly, updates the score, and displays the final result with the highest score saved locally in the browser.

---

- ✅ Responsive UI (Desktop & Mobile)

## ⚙️ How It Works

1. User starts the quiz.
2. A question is displayed.
3. User selects an answer.
4. The application checks the answer.
5. Correct answer is highlighted.
6. Feedback is displayed instantly.
7. Score is updated.
8. Next question is loaded.
9. Final score is shown.
10. Highest score is saved using Local Storage.

---

## 📌 JavaScript Functions

### `loadQuestion()`
- Loads the current question.
- Creates answer buttons dynamically.
- Updates the progress indicator.

### `checkAnswer()`
- Validates the selected option.
- Highlights the correct answer.
- Displays instant feedback.
- Updates the score.

### `showResult()`
- Displays the final score.
- Calculates percentage.
- Saves and displays the highest score.

### `restartQuiz()`
- Resets score and question index.
- Restarts the quiz without refreshing the page.
